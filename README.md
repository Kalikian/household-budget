# Household Budget App

**Live Demo:** https://kalikian.github.io/household-budget/

## Features

- Create entries (income/expense) with amount, category, and optional note
- Entries list rendered instantly after submit (state-driven UI)
- Balance summary based on all entries
- Delete single entries
- Clear all entries
- Persistent data storage via localStorage

## Tech Stack

- React (Vite)
- TypeScript
- Tailwind CSS
- localStorage (persistence)
